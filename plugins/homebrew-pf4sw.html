Adding Homebrew Choices to Quilvyn's PF4SW Rule Set

The PF4SW rule set allows you to add homebrew choices to all of the same types discussed in the SWADE Homebrew Examples document other than era. (The race type is replaced by ancestry in PF4SW.) In addition, PF4SW also allows you to add homebrew deities and languages.
